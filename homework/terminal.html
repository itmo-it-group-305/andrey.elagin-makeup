Шпаргалка по командам терминала

На главную

Ниже приведены команды терминала, которые я черпаю из курса по ОС Linux.

Работа с файлами

pwd
print name of current/working directory
ls
list directory contents (-l вывести в виде списка, -hl список с указанием размера,
-ahl отобразить скрытые файлы и папки)
clear
очистить терминал
exit
выход
mkdir путь_до_директории
(создать директорию)
mkdir -p
создать директорию и ее родителей
touch путь_до_файла
создать файл
cd путь_до_директории
войти в директорию
rm путь_до_файла
удаление файла
rm -r путь_до_директории
удалить директорию
rm -rf путь_до_директории
удалить директорию (без вопросов)
cp путь_до_файла1 путь_до_файла2
скопировать файл1 в файл2
cp -r путь_до_директории1 путь_до_директории2
скопировать дир1 в дир2
mv путь1 путь2
переместить директорию или файл

Запуск и работа с программами в терминале

gedit
как пример, запуск программы gedit
bg
работа с программой в фоновом режиме
fg
работа с программой в терминальном режиме
chmod +x
добавляет свойство executable к программе
&
при добавлении амперсанта к имени программы при запуске, она сразу запускается в фоновом режиме

Просмотр и редактирование файлов

cat имя_файла
запуск файла через терминальное приложение cat
less имя_файла
запуск файла через терминальное приложение less
nano имя_файла
запуск файла через терминальное приложение nano

Перенаправление ввода/вывода

программа < файл
брать stdin из файла
программа > файл
выводить stdout в файл
программа >> файл
работает аналогично с перечисленным выше, только с дозаписью в файл
